Задание которое нужно выполнить:

Организовать общение 2 (и более) приложений через RPC на языке
JavaScript, без использования каких-либо сетевых библиотек реализующие
все это из коробки (пример grpc). Отправляемое сообщение вводится в
консоль приложения отправителя, на стороне приложения получателя
выводится:

1.Текст сообщение;
2.Идентификатор приложения отправителя;
3.Идентификатор приложения получателя (если имеется);
4.Время отправления

Входной параметр:
Идентификатор приложения получателя / -1 (всем);
Текст сообщения;

Дополнение к заданию:
Реализовать end-to-end шифрование. Основная задача заключается в
создание защищённых сообщений, которые может прочитать только
отправитель и получатель.

Пример:
server.js– выступает в роли сервера;
client.js– клиенты которые общаются между собой по аналогии с чатом.
